flex-grow

View SCSS source

Use to apply flex-grow values with our mixin and utility classes.

The flex-grow property determines how much a flex item will grow — relative to the rest of the flex items in a flex container. Flex items will only grow if there’s positively distributed free space in the flex container. Negative numbers are invalid.

http://w3.org/tr/css3-flexbox/#flex-grow-property

This work is derived from sass-flex-mixin.

Values

  • 0 (default)
  • <number>
  • 0 to 10 for use in utility classes
