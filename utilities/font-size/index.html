font-size

View SCSS source

Use font-size to apply values using our mixin or utility classes. Values can optionally be applied only at specific breakpoints.

Using the Sass mixin is preferred, as it forces us to consider all objects as components. Applying utility classes in HTML enforces !important, as they’re intended to be immutable — applying one property and value no matter what.

Values

  • default
  • xx-small
  • x-small
  • small
  • large
  • x-large
  • xx-large
  • 1 to 17 (only available in mixin)

Breakpoints

font-size values can selectively only be applied when certain breakpoints are triggered.

FreeAgent is a company born out of the frustration that business finances are just too damn hard for freelancers and small businesses. 
<div class="u-lg-font-size--x-large u-lg-line-height--tight">
  FreeAgent is a company born out of the frustration that business finances are just too damn hard for freelancers and small businesses.
</div>
.MyComponent { @include break(lg) { @include font-size(x-large); @include line-height(tight); } }
.MyComponent {
  @include break(lg) {
    @include font-size(x-large);
    @include line-height(tight);
  }
}
