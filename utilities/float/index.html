float

View SCSS source

Use to apply float values with utility classes, and ‘clearfix’ hack with a mixin or utility classes. Values can optionally be applied only at specific breakpoints.

Related: clear

Values

  • left
  • none
  • right

Examples

This element floats to the left 
<div class="u-floatContainer">
  <div class="u-background-color--gray-13 u-float--left">
    This element floats to the left
  </div>
</div>
.MyComponent { @include floatContainer; } .MyComponent-child { float: left; }
.MyComponent {
  @include floatContainer;
}

.MyComponent-child {
  float: left;
}
This doesn't float until the medium breakpoint, when it floats to the right 
<div class="u-floatContainer">
  <div class="u-background-color--gray-13 u-md-float--right">
    This doesn't float until the medium breakpoint, when it floats to the right
  </div>
</div>
.MyComponent { @include floatContainer; } .MyComponent-child { @include break(md) { float: right; } }
.MyComponent {
  @include floatContainer;
}

.MyComponent-child {
  @include break(md) {
    float: right;
  }
}
This element would have any inherited float removed 
<div class="u-floatContainer">
  <div class="u-background-color--gray-13 u-float--none">
    This element would have any inherited float removed
  </div>
</div>
.MyComponent { @include floatContainer; } .MyComponent-child { float: none; }
.MyComponent {
  @include floatContainer;
}

.MyComponent-child {
  float: none;
}
